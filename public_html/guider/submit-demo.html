If you are unsure of the photo location there are a number of online sources available to help:

  • OS getamap provides a search by Placename or Postcode.
    Once you have centred the map on the picture location, return here and enter the Grid reference at centre value shown into the box above.

  • multimap.com now displays 1:50,000 Mapping for Northern Ireland. Use our handy Lat/Long Convertor to get the correct Grid Square for a picture.

    Furthermore OSNI and OSI now offer online mapping from their own websites. Coordinate conversion may not be easy - its probably best to rely on visual estimation using the national grid projected on the map.

  • If you have a WGS84 latitude & longitude coordinate (e.g. from a GPS receiver, or from multimap site), then see our Lat/Long to Grid Reference Convertor

  • For information on Grid References
    see Interactive Guide to the National Grid in Great Britain. The Irish National Grid is very similar, but using a single letter prefix, see Overview Map for the layout of the squares.
